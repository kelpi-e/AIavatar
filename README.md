# Видео-аватар на Unreal Engine 5

> **Исходный код** из-за большого объема загружен на Google Drive. Доступен по ссылке: [Скачать исходники](https://drive.google.com/file/d/1cksl8ibCwPveJ9UZRsj60HFPgPiBpFYb/view?usp=sharing)

Интерактивный виртуальный персонаж-помощник с полноценной мимикой, жестами, голосовым и текстовым вводом.

---

## Онлайн-демонстрация

Протестировать приложение можно по ссылке: [Попробовать демо](https://share.arcware.cloud/v1/share-53052340-36dd-412b-a3d3-8170e010b09e)

> **Примечание:** На демо-версии присутствует водяной знак из-за использования бесплатного хостинга. В самом проекте водяного знака нет. Также разрешение картинки в демо ограничено 1920×1080, при локальном запуске разрешение подстраивается под монитор.

## Быстрый запуск (Windows)

1. Перейдите в раздел Releases
2. Скачайте готовую сборку (папка Windows)
3. Распакуйте архив
4. Запустите `fp.exe`

Приложение работает сразу после запуска.

### Протестировано на конфигурациях

**Конфигурация 1:**
- **CPU:** AMD Ryzen 7 5800X
- **GPU:** NVIDIA GeForce RTX 3080 12 GB
- **RAM:** 32 GB
- **OS:** Windows 11 Pro 23H2

**Конфигурация 2:**
- **CPU:** Intel Core i9-13900H (Mobile)
- **GPU:** Integrated
- **RAM:** 16 GB
- **OS:** Windows 11 Pro 24H2

> На более слабых системах возможно снижение производительности.

## Управление и функционал

### Текстовый ввод

1. Нажмите `Enter`
2. Введите сообщение
3. Персонаж сформирует и озвучит ответ

### Голосовой ввод

1. Удерживайте клавишу `T` (английская раскладка)
2. Произнесите запрос

### Поведение персонажа

- Полная лицевая анимация
- Жестикуляция
- Озвучивание ответа
- Роль персонажа: офисный помощник

## Технологии

- **Unreal Engine 5.6.1** — игровой движок
- **MetaHuman** — система создания реалистичных персонажей
- **ConvAI** — плагин для интеграции AI-диалогов (с модификацией для GigaChat)
- **LiveLink** — система синхронизации данных в реальном времени
- **LiveLinkControlRig** — управление анимацией через Control Rig
- **AppleARKitFaceSupport** — поддержка лицевой анимации
- **PixelStreaming** — потоковая передача контента
- **C++** — язык программирования для разработки
